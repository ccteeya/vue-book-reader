# vue-book-reader
用vue.js实现的手机电子书阅读器
## 实现功能
1、电子书的基本阅读功能
2、以及一些辅助功能
- 前后翻页
- 字号设置
- 主题设置
- 进度调整
- 电子书目录
## 涉及技术点
1、web前端基础知识：html,css,javascript基础
2、前端框架 vue.js 使用：vue基础语法，vue-cli项目搭建，vue-router, vue组件化开发, 父子组件通信，dom操纵, 过渡动画实现
3、scss: 基本语法，@import, @mixin
4、了解epub解析原理

## 本地运行

```javascript
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
```
